# Porter

A small utility program for bridging WebSocket connections, useful when you need to forward messages from a local WebSocket server to a remote one.

## Requirements

- [Go](https://go.dev)

## Build

```sh
$ make all
```

## Usage

```sh
$ porter-windows-amd64.exe ws://localhost:1234 wss://my.server.dev
```
